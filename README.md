# 🚀 Startup Arena! 

Um torneio de startups com backend em Node.js/Express e banco de dados em PostgreSQL.

---

### 🛠 Pré‑requisitos
- Docker ≥ 20.x  
- Docker Compose ≥ 1.29.x  


---

### 🐳 Rodando com Docker
1. **Build & up**  
   docker-compose up --build

2. **Verificar**  
   abra http://localhost:3000
